Laundry Rules

These rules explain how laundry records work on this website.

Recording laundry

  1. Record the laundry after starting it.
  2. The server records the time; the time on the visitor's device is not used.
  3. Two laundry records cannot be created within one hour of each other.
  4. The goal is to record at least one laundry during every rolling 40-hour period.

Deleting a record

  1. A record can be deleted only when it is less than one hour old.
  2. The website asks for confirmation before deleting a record.
  3. After one hour, the Delete button is removed and the server rejects deletion attempts.

History and access

  1. The history displays the 20 most recent laundry records, newest first.
  2. Displayed times use the visitor's local time zone.
  3. Anyone with access to this website can record laundry and delete eligible records.
