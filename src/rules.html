Home Task Rules

These rules explain how task records work on this website.

Tasks

  1. Record laundry after starting it.
  2. Record guinea pig cleaning after cleaning the guinea pigs.
  3. Each task has its own rolling 40-hour target.

Recording a task

  1. The server records the time; the time on the visitor's device is not used.
  2. Two records for the same task cannot be created within one hour of each other.
  3. Recording one task does not prevent the other task from being recorded.

Deleting a record

  1. A record can be deleted only when it is less than one hour old.
  2. The website asks for confirmation before deleting a record.
  3. After one hour, the Delete button is removed and the server rejects deletion attempts.

History and access

  1. Each history displays its 20 most recent records, newest first.
  2. Displayed times use the visitor's local time zone.
  3. Anyone with access to this website can record tasks and delete eligible records.
